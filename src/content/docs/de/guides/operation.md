Diese Anleitung beschreibt die Bedienung des BGF Ü-Streaming (Übersetzungs-Streaming) für IGR-Konferenzen.

## Vorbereitung

### Vor der Konferenz

1. **Hardware-Check**: Stellen Sie sicher, dass die Hardware gemäß [Hardware-Setup](/de/advanced/hardware-setup/) aufgebaut ist
2. **Netzwerk-Test**: Überprüfen Sie die Internetverbindung
3. **Software-Start**: Starten Sie die Streaming-Software
4. **Audio-Test**: Testen Sie alle Audiokanäle

### Checkliste

- [ ] Hardware aufgebaut und verkabelt
- [ ] Netzwerkverbindung stabil
- [ ] Streaming-Software gestartet
- [ ] Alle Audiokanäle funktionieren
- [ ] Übersetzer sind bereit

## Während der Konferenz

### Streaming starten

1. Öffnen Sie die Streaming-Software
2. Wählen Sie die aktive Konferenz aus
3. Aktivieren Sie die benötigten Sprachkanäle (siehe [Streamplan](/de/2025-swo/streamplan/))
4. Klicken Sie auf "Stream starten"

### Sprachen verwalten

- **Sprache aktivieren**: Klicken Sie auf den entsprechenden Sprachkanal
- **Sprache deaktivieren**: Klicken Sie erneut auf den Kanal
- **Lautstärke anpassen**: Verwenden Sie die Lautstärkeregler für jeden Kanal

### Überwachung

Während des Streamings sollten Sie kontinuierlich überwachen:

- **Audio-Pegel**: Stellen Sie sicher, dass die Pegel im grünen Bereich sind
- **Netzwerk-Status**: Überprüfen Sie die Verbindungsqualität
- **Teilnehmer-Feedback**: Achten Sie auf Rückmeldungen von Teilnehmern

## Troubleshooting

### Kein Audio

1. Überprüfen Sie die Kabelverbindungen
2. Stellen Sie sicher, dass die Mikrofone eingeschaltet sind
3. Prüfen Sie die Audio-Einstellungen in der Software

### Verbindungsprobleme

1. Überprüfen Sie die Netzwerkverbindung
2. Starten Sie den Router neu
3. Kontaktieren Sie den IT-Support

### Audio-Verzögerung

1. Reduzieren Sie die Streaming-Qualität
2. Überprüfen Sie die Netzwerkbandbreite
3. Schließen Sie andere Anwendungen

## Nach der Konferenz

1. Stoppen Sie alle Streams
2. Speichern Sie die Logs (falls erforderlich)
3. Fahren Sie die Hardware ordnungsgemäß herunter
4. Dokumentieren Sie eventuelle Probleme
