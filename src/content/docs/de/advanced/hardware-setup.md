Diese Anleitung beschreibt den Aufbau der Hardware für das BGF Ü-Streaming vor IGR-Konferenzen.

## Benötigte Hardware

### Hauptkomponenten

- **Streaming-Server**: Computer oder Laptop mit ausreichender Leistung
- **Audio-Interface**: Für den Empfang der Übersetzungskanäle
- **Netzwerk-Equipment**: Router, Switches, Kabel
- **Mikrofone**: Für Übersetzer (falls erforderlich)
- **Kopfhörer**: Für Übersetzer

### Kabel und Zubehör

- Ethernet-Kabel (Cat6 oder besser)
- Audio-Kabel (XLR, Klinke)
- Stromkabel und Verlängerungen
- Adapter (falls erforderlich)

## Aufbau-Anleitung

### Schritt 1: Standort vorbereiten

1. Wählen Sie einen geeigneten Standort für die Hardware
2. Stellen Sie sicher, dass ausreichend Steckdosen vorhanden sind
3. Überprüfen Sie die Netzwerkanbindung

### Schritt 2: Netzwerk einrichten

1. Positionieren Sie den Router zentral
2. Verbinden Sie den Router mit dem Internet
3. Schließen Sie den Streaming-Server per Ethernet an
4. Testen Sie die Internetverbindung

### Schritt 3: Audio-Equipment anschließen

1. Verbinden Sie das Audio-Interface mit dem Streaming-Server
2. Schließen Sie die Mikrofone an das Audio-Interface an
3. Verbinden Sie die Kopfhörer für die Übersetzer
4. Testen Sie alle Audio-Verbindungen

### Schritt 4: Software konfigurieren

1. Installieren Sie die Streaming-Software (falls noch nicht geschehen)
2. Konfigurieren Sie die Audio-Eingänge
3. Richten Sie die Sprachkanäle ein
4. Testen Sie die Konfiguration

## Verkabelungsplan

```

         |

         +--- Mikrofon 3 (Französisch)
         +--- ...
```

## Checkliste vor der Konferenz

- [ ] Alle Geräte sind angeschlossen und eingeschaltet
- [ ] Netzwerkverbindung ist stabil
- [ ] Audio-Kanäle funktionieren
- [ ] Software ist konfiguriert und getestet
- [ ] Backup-Equipment ist verfügbar
- [ ] Kontaktdaten für technischen Support sind vorhanden

## Sicherheitshinweise

- Verwenden Sie Kabelkanäle oder Klebeband, um Stolperfallen zu vermeiden
- Sichern Sie alle Verbindungen gegen versehentliches Lösen
- Halten Sie Backup-Kabel und -Geräte bereit
- Dokumentieren Sie die Verkabelung für spätere Referenz

## Troubleshooting

### Keine Netzwerkverbindung

1. Überprüfen Sie alle Ethernet-Kabel
2. Starten Sie den Router neu
3. Überprüfen Sie die Netzwerkeinstellungen am Server

### Audio-Probleme

1. Überprüfen Sie alle Audio-Verbindungen
2. Testen Sie die Mikrofone einzeln
3. Überprüfen Sie die Audio-Interface-Einstellungen

### Stromausfall

1. Verwenden Sie eine USV (Unterbrechungsfreie Stromversorgung)
2. Halten Sie Backup-Batterien bereit
3. Dokumentieren Sie die Stromversorgung

## Abbau nach der Konferenz

1. Fahren Sie alle Geräte ordnungsgemäß herunter
2. Trennen Sie alle Verbindungen systematisch
3. Verpacken Sie die Hardware sorgfältig
4. Dokumentieren Sie eventuelle Schäden oder Probleme
