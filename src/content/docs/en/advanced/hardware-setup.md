This guide describes the hardware setup for BGF Translation Streaming before IGR conferences.

## Required Hardware

### Main Components

- **Streaming Server**: Computer or laptop with sufficient performance
- **Audio Interface**: For receiving translation channels
- **Network Equipment**: Router, switches, cables
- **Microphones**: For translators (if required)
- **Headphones**: For translators

### Cables and Accessories

- Ethernet cables (Cat6 or better)
- Audio cables (XLR, jack)
- Power cables and extensions
- Adapters (if required)

## Setup Instructions

### Step 1: Prepare Location

1. Choose a suitable location for the hardware
2. Ensure sufficient power outlets are available
3. Check network connectivity

### Step 2: Set Up Network

1. Position the router centrally
2. Connect the router to the internet
3. Connect the streaming server via Ethernet
4. Test the internet connection

### Step 3: Connect Audio Equipment

1. Connect the audio interface to the streaming server
2. Connect microphones to the audio interface
3. Connect headphones for translators
4. Test all audio connections

### Step 4: Configure Software

1. Install streaming software (if not already done)
2. Configure audio inputs
3. Set up language channels
4. Test the configuration

## Pre-Conference Checklist

- [ ] All devices are connected and powered on
- [ ] Network connection is stable
- [ ] Audio channels are working
- [ ] Software is configured and tested
- [ ] Backup equipment is available
- [ ] Technical support contact details are available

## Safety Notes

- Use cable channels or tape to avoid tripping hazards
- Secure all connections against accidental disconnection
- Keep backup cables and devices ready
- Document the wiring for later reference

## Troubleshooting

### No Network Connection

1. Check all Ethernet cables
2. Restart the router
3. Check network settings on the server

### Audio Problems

1. Check all audio connections
2. Test microphones individually
3. Check audio interface settings

### Power Failure

1. Use a UPS (Uninterruptible Power Supply)
2. Keep backup batteries ready
3. Document power supply

## Teardown After Conference

1. Shut down all devices properly
2. Disconnect all connections systematically
3. Pack hardware carefully
4. Document any damage or issues
