This guide describes how to operate BGF Translation Streaming for IGR conferences.

## Preparation

### Before the Conference

1. **Hardware Check**: Ensure hardware is set up according to [Hardware Setup](/en/advanced/hardware-setup/)
2. **Network Test**: Verify internet connection
3. **Software Start**: Launch streaming software
4. **Audio Test**: Test all audio channels

### Checklist

- [ ] Hardware assembled and wired
- [ ] Network connection stable
- [ ] Streaming software started
- [ ] All audio channels working
- [ ] Translators are ready

## During the Conference

### Starting the Stream

1. Open the streaming software
2. Select the active conference
3. Activate required language channels (see [Stream Schedule](/en/2025-swo/streamplan/))
4. Click "Start Stream"

### Managing Languages

- **Activate language**: Click on the corresponding language channel
- **Deactivate language**: Click the channel again
- **Adjust volume**: Use volume controls for each channel

### Monitoring

During streaming, continuously monitor:

- **Audio levels**: Ensure levels are in the green range
- **Network status**: Check connection quality
- **Participant feedback**: Pay attention to feedback from participants

## Troubleshooting

### No Audio

1. Check cable connections
2. Ensure microphones are turned on
3. Check audio settings in software

### Connection Problems

1. Check network connection
2. Restart the router
3. Contact IT support

### Audio Delay

1. Reduce streaming quality
2. Check network bandwidth
3. Close other applications

## After the Conference

1. Stop all streams
2. Save logs (if required)
3. Shut down hardware properly
4. Document any issues
