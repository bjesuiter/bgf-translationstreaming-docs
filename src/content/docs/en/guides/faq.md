Here you'll find answers to the most frequently asked questions about BGF Translation Streaming for IGR conferences.

## General Questions

### What is BGF Translation Streaming?

BGF Translation Streaming is the system for live translations at IGR conferences. It allows participants to receive simultaneous translations in various languages through their own devices.

### Which languages are supported?

Available languages depend on the specific conference. Typically, German, English, French, Spanish, and additional languages are offered. Details can be found in the [Stream Schedule](/en/2025-swo/streamplan/) for each conference.

## Operation

### How do I start Translation Streaming?

A detailed guide can be found in the [Operation Guide](/en/guides/operation/).

### What should I do if technical problems occur during the conference?

1. Check the network connection
2. Ensure all devices are powered on
3. Contact on-site technical support
4. Consult the [Operation Guide](/en/guides/operation/) for troubleshooting

### How do I switch between different languages?

Language switching is done through the streaming software. Details can be found in the [Operation Guide](/en/guides/operation/).

## Hardware

### What hardware is required?

A complete list of required hardware and setup instructions can be found in [Hardware Setup](/en/advanced/hardware-setup/).

### When should the hardware be set up?

Hardware should be set up and tested at least one day before the conference. Details in [Hardware Setup](/en/advanced/hardware-setup/).
